# Racks
**Racks** is a prototype of a space accounting system for telecommunication cabinets and racks.

**Tech stack:**
- Python 3.10
- Django 2.2
- Django REST framework 0.1
- Vue 3.2
- Tailwindcss 3.2
- Gunicorn 19.9
- PostgreSQL 10.18
