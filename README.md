# Racks
**Racks** is a prototype of a space accounting system for telecommunication cabinets and racks.

**Stack:**
- Python 3.10
- Django 2.2
- Django REST framework 3.13
- Vue 3.2
- TailwindCSS 3.2
- PostgreSQL 10.18
